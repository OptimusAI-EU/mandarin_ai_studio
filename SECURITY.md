# Security

Open Video Studio is designed to run locally on your machine.

## API Keys

The OpenRouter API key is stored locally in `.ovstudio/studio.sqlite`. Do not commit `.ovstudio/`, and do not deploy this app publicly with a saved API key.

## Reporting Issues

Please report security issues privately to the project maintainers. If the repository does not yet list a security contact, open a minimal public issue asking for a private disclosure channel without including exploit details.

## Supported Versions

This project is pre-1.0. Security fixes are made on the latest main branch.
